◆ flash_exec()

int flash_exec ( char *  fname)

This function copies a file from the sd card to flash memory and executes it.

Parameters
fnameThe filename to execute
Returns
on success it doesn't return
on failure it returns the following codes:
EXEC_NO_CARD – No SD card detected
EXEC_CANNOT_MOUNT – Cannot mount FAT filesystem
EXEC_FILE_NOT_FOUND – file not found in FAT filesystem
EXEC_BAD_SIZE – File not expected size (262144 bytes)
EXEC_INTERNAL_ERROR – Internal error occurred

◆ flash_rollback()

void flash_rollback ( void  )

rollback the current program memory into the flash

This function programs the FT900 flash with the current FT900 data memory. It is primarily to restore the flash memory when there are issues when programming the flash, causing the flashing process to be aborted.
